# NFT-Game
# Description
NOMBRE: Find It Out.

TIPO: Estrategia, NFT, play to earn.

ABOUT THE GAME: Find It Out es un juego de cartas y estrategia en el cual los jugadores tendran que descubrir el misterio que hay detras. Se puede jugar con un minimo de 2 jugadores hasta un maximo de 6. Se seleccionan 3 cartas al azar que los participantes deberan descubrir, el resto de cartas se reparten equitativamente entre los jugadores.
